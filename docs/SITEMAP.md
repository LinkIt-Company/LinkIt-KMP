## 전체 구조 개요

```
앱 실행
├── [최초 진입 여부 판단]
│   ├── Y (첫 방문) → 온보딩 Flow
│   └── N (재방문) → 메인화면 (지도탭)
│
├── 메인화면 (지도탭)          ← Bottom Nav
├── 탐색 & 피드               ← Bottom Nav (pending)
└── 보관함                    ← Bottom Nav (pending)
```

## 1. 온보딩 Flow (최초 진입)

```
최초진입? ──Y──→ 인트로 → 서비스 / 일정 생성 작동방식 설명
                           │
                           ├─ [일정생성] → "영상 링크로 만들기" 를 선택해 보세요
                           │    ├─ 영상 링크로 만들기  ← 이 버튼만 활성화
                           │    ├─ 보관함에서 가져오기  ← 선택 불가
                           │    └─ 직접 만들기          ← 선택 불가
                           │
                           └──→ 일정생성 상세 페이지 → 추천영상 확인 → 링크복사
                                                       → 복사한 링크 붙여넣기
                                                       → 로딩 팝업
                                                       → 생성된 일정 확인
                                                            └──→ 일정 저장 Flow
```

### 일정 저장 Flow (공통)

생성된 일정 확인 화면, 영상 요약탭, 여행 일정탭에서 공통으로 사용되는 저장/삭제 흐름:

```
[저장] ──→ 전체일정 저장하기 → 저장 폴더 설정 페이지 → 일정 저장완료
       └─→ 선택항목 저장하기 → 일정 선택 페이지       → 선택항목 저장완료

[삭제] ──→ 삭제 팝업 생성 ("이 일정을 정말 삭제하시겠어요?")
```

**생성된 일정 확인 페이지 상세 내용:**
- 요약된 영상 정보
- 일정 / 예산
- 일차별 스케줄
- 함께 볼만한 영상
- 일정 만들기

## 2. 메인화면 (지도탭)

앱의 핵심 화면. 지도 위에 저장된 일정이 마커로 표시된다.

```
메인화면 (지도탭)
│
├── 일정 (바텀시트)
│   ├── 국가 필터 (일본, 베트남 등) → 저장된 일정의 국가에 따라 생성
│   ├── [일정 선택] → 일정 상세페이지 (A)
│   └── [필터] → 지역 필터 / 여행 테마 필터 / 비용 필터
│
├── [프로필] → 프로필 페이지
│   ├── [관심사 설정] → 선호하는 스타일 / 여행지 등
│   └── [앱 초기화]
│
├── [검색] → 검색 페이지
│   ├── 최근 검색어 / 유사 검색어 (추천 검색어)
│   ├── [직접 입력]
│   └── → 검색 결과
│
├── [지도뷰 변경] → 기본값 / 위성뷰 전환
│
├── [현재 위치] → 위치권한 허용 여부 판단
│   ├── Y → 현재 위치 이동 (GPS로 지도 이동)
│   └── N → 위치권한 팝업 생성
│
├── [일정 생성] → 일정 생성 Flow (아래 참조)
│
└── [마커 / 영역 선택] → 일정 상세페이지
                          (해당 마커가 속한 일정 상세페이지로 이동)
```

### 메인화면에서의 일정 생성 Flow

```
[일정 생성] → 영상 링크로 만들기
             → 보관함에서 가져오기
             → 직접 만들기

         └──→ [일정 생성하기]
              → 일정생성 상세 페이지 → 로딩 팝업 → 생성된 일정 확인
                                                     └──→ 일정 저장 Flow (공통)
```

- 추천영상: 일정생성 상세 페이지 하단에 표시

---

## 3. 일정 상세페이지

바텀시트 일정 선택 또는 마커 선택 시 진입하는 상세 화면.

```
일정 상세페이지
│
├── 해당 일정의 위치로 지도 이동
├── 도시 필터 / 테마 필터 / 비용 필터
│
├── [일정 선택] → 특정 일정 상세
│
├── 영상 요약탭
│   ├── 요약된 영상정보
│   ├── 여행 기간
│   ├── 여행 스타일
│   ├── 방문 장소 등
│   └── 일정 저장 Flow (공통)
│
└── 여행 일정탭
    ├── 날짜별 여행지 및 교통
    └── 일정 저장 Flow (공통)
```
