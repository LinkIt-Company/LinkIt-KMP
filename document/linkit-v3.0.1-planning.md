## 1. 앱 개요

**앱 이름:** LinkIt
**핵심 컨셉:** 유튜브 여행 영상 링크를 기반으로 AI가 여행 일정을 자동 생성해주는 여행 플래닝 앱
**플랫폼:** KMP (Kotlin Multiplatform) - iOS/Android

## 2. 화면 구성 (섹션별)

### 2.1 인트로 / 최초 진입 (온보딩)

| 화면 | 설명 |
|------|------|
| **인트로 애니메이션** | 첫 진입 시 지구가 줌인되면서 내 위치로 확장. 위치 권한 미허용 시 서울(한국)로 기본 설정 |
| **온보딩 04** | "유튜브 영상 링크를 넣고 여행 일정을 만들어보세요" - 유튜브 링크 입력 화면. 복사한 링크 붙여넣기 버튼, 폴더명 입력, 같은 이름 폴더 중복 체크 |
| **온보딩 05** | 링크 붙여넣기 확인 화면. 유효하지 않은 영상 링크 에러 메시지 ("유효한 영상 링크가 아닙니다") |
| **온보딩 13~14** | 일정 생성 로딩 화면 - "일정 생성중...", 프로그레스 바, "일정을 생성중이에요~", "완료되면 알려드릴게요~" |

**인트로 관련 기획 메모:**
- 재설치 사용자 고려 필요 → 인트로 건너뛰기 옵션 검토
- 인트로 과정이 길면 이탈 우려 → 직접 써보게끔 설계

### 2.2 메인화면 / 지도

| 화면 | 설명 |
|------|------|
| **지도 탐색 01** | 메인 지도 화면. 저장된 일정이 마커로 표시됨. 하단 네비게이션 바 (지도, 탐색, 보관함, 일정생성) |
| **지도 탐색 7** | 지도 상세/필터 적용 상태 |

**지도 기능:**
- 지도 뷰: 기본값 / 위성뷰 전환
- 저장된 일정이 마커로 표시
- 국가 필터 (일본, 베트남 등 - 저장된 일정의 국가에 따라 동적 생성)
- 지역 필터 / 여행 테마 필터 / 비용 필터
- 마커 선택 시 해당 일정 상세페이지로 이동
- 현재 위치 GPS (위치 권한 팝업)

**하단 네비게이션 바:**
- 지도
- 탐색
- 보관함
- 일정 생성

### 2.3 일정 생성 Flow

**일정 생성 방법 3가지:**
1. **영상 링크로 만들기** (핵심 기능, 최초 진입 시 이것만 선택 가능)
2. 보관함에서 가져오기
3. 직접 만들기

**영상 링크 일정 생성 프로세스:**
1. 일정 생성 버튼 선택
2. "영상 링크로 만들기" 선택 (나머지 버튼은 초기에 선택 불가)
3. 링크 복사해서 입력 (클립보드 복사 링크 팝업 자동 생성)
4. AI 분석 시작 → 로딩 화면
5. 분석 완료 → 일정 결과 표시

**AI 분석 결과 화면:**
- AI 요약 정보 (영상 내용 요약)
- 관련 태그 (#도쿄여행, #먹방브이로그 등)
- 여행 정보: 여행 기간, 예상 비용, 여행 스타일
- 영상 타임라인 (시간대별 구간 정보)
- 일차별 스케줄 (장소, 교통, 소요시간)
- 장소 상세 정보 (평점, 리뷰, 운영시간, 입장료 등)
- 함께 볼만한 영상 추천

**일정 저장:**
- "전체일정 저장하기" 또는 "선택항목 저장하기"
- 저장할 폴더 선택 팝업
- 일정 삭제 확인 ("이 일정을 정말 삭제하시겠어요?")

### 2.4 탐색 & 피드

| 섹션 | 내용 |
|------|------|
| **인기 여행지 둘러보기** | "다음에 어떤 여행을 떠나볼까요?" - Tokyo (1.2k+ videos), Bangkok (800+ videos), Rome (650+ videos) |
| **국가별 인기 영상** | "일본에서 인기있는 여행 영상" - 한국 사람들이 많이 분석한 관련 영상 리스트 |
| **인기 크리에이터** | 여행 크리에이터 프로필 (빠니보틀 240만 팔로워 등) + "영상 보러가기" |
| **테마별 추천 영상** | 설정된 선호도에 따른 추천 - 미식 여행, 힐링 여행, 도심지 여행, 자연속 여행 |
| **크리에이터별 영상** | Sam the Nomad (Tokyo travel expert), Elena Wander (European city guide) 등 |

**탐색 필터:**
- 국가 필터: 전체, 일본, 미국, 호주, 중국 등
- 검색: "검색어를 입력해주세요"

### 2.6 보관함

**2가지 UI 안 검토 중:**

| 안 | 설명 |
|------|------|
| **A안** | 보관된(저장된) 항목만 리스트로 보여주는 안 |
| **B안** | 지도와 함께 보여지는 안 |

**보관함 기능:**
- 저장 항목 추가하기 (직접 추가)
- 일차별 일정 관리 (1일차 ~ 5일차)
- 장소 정보 표시 (시간, 장소명, 주소, 메모)
- AI Analysis Insights (검증률 표시 "Verified 98%")
- 추천 영상 둘러보기

## 3. 사이트맵 (화면 흐름)

```
앱 실행
├─ [Y: 최초진입] → 인트로 애니메이션 → 온보딩 Flow
│   └─ 일정 생성 버튼 → 영상 링크로 만들기 → 링크 입력 → 폴더명 입력
│       → AI 분석 → 일정 결과 → 저장
│
├─ [N: 재방문] → 메인 지도 화면
│   ├─ 지도 (메인)
│   │   ├─ 국가별/지역별 필터
│   │   ├─ 일정 마커 → 일정 상세
│   │   ├─ 기본값/위성뷰 전환
│   │   └─ 현재 위치 GPS
│   │
│   ├─ 탐색
│   │   ├─ 인기 여행지
│   │   ├─ 인기 크리에이터
│   │   └─ 테마별 추천 영상
│   │
│   ├─ 보관함
│   │   ├─ 저장된 일정 리스트
│   │   ├─ 일차별 관리
│   │   └─ 장소 상세 (리뷰, 운영시간 등)
│   │
│   └─ 마이페이지
│       ├─ 지도 설정
│       ├─ 도움말 & 지원
│       └─ 앱 설정 (초기화)
│
├─ 일정 상세 페이지
│   ├─ 요약된 영상 정보 / 여행 기간 / 여행 스타일
│   ├─ 날짜별 여행지 및 교통
│   ├─ 장소 상세 (평점, 리뷰, 사진, 운영시간, 입장료)
│   ├─ 일정 삭제 확인 팝업
│   └─ 함께 볼만한 영상 / 일정 만들기
│
└─ 공통 UI
    ├─ 선호 스타일 설정 (여행지, 테마 등)
    ├─ 폴더 선택 팝업
    └─ 토스트 메시지
```

## 4. 핵심 기능 요약

| # | 기능 | 우선순위 | 설명 |
|---|------|----------|------|
| 1 | 유튜브 링크 기반 AI 일정 생성 | 핵심 | 유튜브 영상 링크 → AI 분석 → 여행 일정 자동 생성 |
| 2 | 지도 기반 일정 관리 | 높음 | 저장된 일정을 지도 마커로 시각화, 필터링 |
| 3 | 일정 저장 & 폴더 관리 | 높음 | 전체/선택 저장, 폴더 분류 |
| 4 | 장소 상세 정보 | 중간 | 평점, 리뷰, 운영시간, 입장료, 주변 정보 |
| 5 | 탐색 & 피드 | 중간 | 인기 여행지, 크리에이터, 테마별 추천 |
| 6 | 일정 생성 flow (pending) | 보류 | 사이트맵에 "일정 생성 flow" 별도 표시, 일부 pending 상태 |

## 5. 기획 이슈 / 미결정 사항

1. **인트로 건너뛰기** - 재설치 사용자 대응, 실수로 건너뛰기 선택 시 대응 방안 미정
2. **보관함 UI** - A안(리스트형) vs B안(지도 병행형) 최종 결정 필요
3. **일정 생성 방법 2, 3** - "보관함에서 가져오기", "직접 만들기"는 초기에 비활성 → 향후 활성화 시점 미정
4. **일정 생성 flow** - 사이트맵에 "pending" 라벨로 별도 표기, 상세 플로우 확정 필요
